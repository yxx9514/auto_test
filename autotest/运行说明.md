## 自动化测试项目 - 运行说明（pytest + allure）

### 1) 安装（建议独立虚拟环境）

在 `autotest/` 目录执行：

```powershell
py -m venv .venv
.\.venv\Scripts\activate
py -m pip install -U pip
pip install -e .
```

### 2) 运行测试

```powershell
pytest
```

仅跑某一类：

- API：`pytest -m api`
- 关键字：`pytest -m keyword`
- UI：`pytest -m ui`

### 3) 生成 Allure 报告

前提：本机已安装 Allure CLI（`allure` 命令可用）

```powershell
allure serve allure-results
```

### 4) UI 自动化（Playwright，可选）

如果要跑 UI 用例：

```powershell
pip install -e ".[ui]"
playwright install
pytest -m ui
```
