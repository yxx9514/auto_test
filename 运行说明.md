# 量化炒股辅助软件 - 运行说明

## 这是一个 Streamlit Web 应用

这个应用需要在浏览器中运行，不能直接用 `python transaction_review.py` 运行。

## 运行方法

### 方法1：使用命令行运行（推荐）

1. 打开终端/命令行
2. 进入项目目录：
   ```
   cd D:\learn\project\pythonProject\PythonProject2
   ```
3. 激活虚拟环境（Windows）：
   ```
   .\venv\Scripts\activate
   ```
4. 运行 Streamlit 应用：
   ```
   streamlit run transaction_review.py
   ```
5. 浏览器会自动打开，如果没有自动打开，请访问终端中显示的URL（通常是 http://localhost:8501）

### 方法2：使用 PowerShell

在项目目录下执行：
```powershell
.\venv\Scripts\streamlit.exe run transaction_review.py
```

## 使用说明

1. 应用启动后，会在浏览器中显示一个Web界面
2. 在左侧边栏输入股票代码（如：600703、600519）
3. 点击回车或等待自动更新
4. 页面会显示：
   - 四个信号灯（趋势因子、量能因子、中期因子、短期因子）
   - 风险预警信息
   - K线图和均线图
   - RSI指标图

## 功能特点

- 📊 实时股票数据分析
- 🔴🟢 四因子信号灯系统（三龙聚首）
- ⚠️ 风险预警（震荡预警、顶背离预警）
- 📈 可视化K线图和指标图
- 🎯 量化交易信号生成

## 注意事项

- 首次运行需要下载股票数据，可能需要几秒钟
- 确保网络连接正常（需要访问akshare数据源）
- 股票代码格式：6位数字（如 600519、000001、300750）
